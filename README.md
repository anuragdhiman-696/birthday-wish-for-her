# 🎉 Birthday Wishes Website

A beautiful and interactive birthday wishes website built using HTML, CSS, and JavaScript. The website features animations, music, heartfelt wishes, and a responsive design to make someone's birthday extra special.

## ✨ Features

- 🎂 Beautiful birthday landing page
- 🎈 Smooth animations and transitions
- 🎵 Background birthday music
- 📱 Responsive design for mobile and desktop
- ❤️ Personalized birthday message
- 🖼️ Custom images and decorations

## 🛠️ Built With

- HTML5
- CSS3
- JavaScript

or use the **Live Server** extension in VS Code.
